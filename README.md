# TaskCRUD-Front

To install dependencies, run
``` bash
npm install
```

## Start local server
1) Copy .env.example to .env and fill it.

2)Start server: 
``` bash
npm run dev
```

## For the test 
  1) Create a test database (do not use the same as production)
  2) Start test ( it uses cypress):
  ``` bash
npm run test
```
